# Серверная версия — запуск

- **Node.js** не ниже 18 (нужен встроенный `fetch`).
- Установка: `npm install`
- Запуск: `npm start` (порт по умолчанию 3000).

## Переменные окружения

- `PORT` — порт (по умолчанию 3000).
- `APPS_SCRIPT_URL` — URL развёрнутого Google Apps Script (тот же, что в config.js для ветки main). Без него кнопка «Обновить данные» и cron не смогут подтянуть данные.

Пример при запуске:

```bash
APPS_SCRIPT_URL="https://script.google.com/macros/s/.../exec" node server.js
```

Данные кэшируются в файле `data.json` в корне проекта. Обновление: 5 раз в сутки по расписанию (00:00, 05:00, 10:00, 15:00, 20:00) и по кнопке «Обновить данные» на дашборде.
